# 🤖 Basic Chatbot

A simple **rule-based chatbot** developed using **Python** as part of the **CodeAlpha Python Programming Internship**.

## 📌 Project Overview

This chatbot interacts with users through the command line by responding to predefined messages such as greetings, questions, and farewells. It demonstrates the use of conditional statements, loops, functions, and user input handling in Python.

---

## ✨ Features

- 👋 Greets the user
- 💬 Responds to predefined questions
- 🔄 Continuous conversation until the user exits
- 💻 Console-based application
- 🚀 Beginner-friendly Python project

---

## ▶️ How to Run

1. Install Python 3.
2. Open the project folder in Visual Studio Code.
3. Open the terminal.
4. Run:

```bash
python chatbot.py
```
